# Barkway

**Smart, simple booking software built by groomers, for groomers.**

Barkway is a specialist software platform for dog grooming businesses. It is designed to help groomers run a more organised, professional business with less admin and a better client experience.

## What Barkway does

Barkway brings together the core tools grooming businesses rely on day to day, including:

- Booking and appointment management
- Client and pet records
- Deposits, payments, and invoicing
- Reminders and recalls
- Reporting and business insights
- Client self-service through the portal

## Why Barkway exists

Generic scheduling tools rarely fit grooming properly. Barkway is built around real grooming workflows, with the goal of making booking, record-keeping, communication, and payments simpler for the businesses using it.

We focus on software that is:

- Practical
- Easy to use
- Built for the realities of pet grooming
- Designed to help businesses operate more professionally

## Open source

We publish selected open source packages and developer tools built as part of Barkway.

These projects usually come from real product needs - things we wanted to build properly for our own platform, then release in a form that is useful to other developers too. We aim to keep them practical, lightweight, and easy to integrate.
